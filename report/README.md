# LaTeX report

The course report source is `course_report.tex`.

Compile from the repository root with XeLaTeX:

```powershell
xelatex -output-directory=report report/course_report.tex
xelatex -output-directory=report report/course_report.tex
```

1. Fill in both members' names, student IDs, and division of work.
2. After the 2026-06-01 to 2026-06-12 simulation period, replace the reserved simulation-trading placeholders with the actual return analysis and screenshots.
3. Keep the figure paths relative to the repository root unless the report is reorganized.
